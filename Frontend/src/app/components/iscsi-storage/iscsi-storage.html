Attach a volume to this VM

Agent is ready.

Your VM is now registered. Here's how to attach a volume:

1
Create a volume using the panel on the left (just name + size). Wait for the status to show Ready.
2
Click "Attach to VM" on the volume card. Select this VM from the list.
3
Within ~10 seconds the disk appears on your VM:
lsblk # new /dev/sdX
4
Format and mount: 
sudo mkfs.ext4 /dev/sdX
sudo mkdir /mnt/data
sudo mount /dev/sdX /mnt/data
